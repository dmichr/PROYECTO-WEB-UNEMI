1 Distribución normal

La distribución normal, también conocida como gaussiana o de Gauss, es una de las distribuciones de probabilidad más comunes en estadística y probabilidad. Su gráfica de densidad tiene una forma acampanada y simétrica en relación con un parámetro estadístico. Se utiliza ampliamente en la modelización de diversos fenómenos naturales, sociales y psicológicos. La distribución normal es fundamental en la estimación por mínimos cuadrados, un método simple y antiguo de estimación.

Si X sigue una distribución normal \(N(μ,σ)\), entonces:

  • \(f(x)\) = \(dnorm(x,mu,sigma)\)

  • \(P(X≤k)\)= \(pnorm(x,mu,sigma)\)

  • \(q_a = min{x:P(X≤x)≥a} = qnorm(a,mu,sigma)\)

  • \(rnorm(n,mu,sigma)\) genera \(n\) valores aleatorios \(N(μ,σ)\)

Supongamos que \(X≈N(170,12)\). Entonces:

  • Calculamos \(f(178)\)
dnorm(171,170,12)

1.1 Función de Densidad

La función de densidad de la distribución normal está dada por: \[ f(x|μ,σ^2) = \frac{1}{σ\sqrt{2\pi}} e^{-\frac{1}{2}\left(\frac{x-μ}{σ}\right)^2} \] donde:

  • \(μ\) es la media de la distribución.

  • \(σ^2\) es la varianza de la distribución (o el cuadrado de la desviación estándar).

  • \(x\) es la variable aleatoria.

Representación gráfica de la funcion de densidad

curve(dnorm(x,170,12),xlim=c(130,210),col="blue",lwd=2,
      xlab="x",ylab="f(x)",main="Función de Densidad N(170,12)")

1.2 Función de Distribución

La función de densidad de la distribución normal está dada por:

\[ f(x|μ,σ^2) = \frac{1}{σ\sqrt{2\pi}} e^{-\frac{1}{2}\left(\frac{x-μ}{σ}\right)^2} \] y la función de distribución acumulada está dada por: \[ F(x|μ,σ^2) = \int_{-\infty}^{x} f(t|μ,σ^2) dt = \frac{1}{2}\left[1 + \text{erf}\left(\frac{x-μ}{σ\sqrt{2}}\right)\right] \]

donde: * \(μ\) es la media de la distribución. * \(σ^2\) es la varianza de la distribución (o el cuadrado de la desviación estándar). * \(x\) es la variable aleatoria. * \(\text{erf}(z)\) es la función error.

Representación de la funcion de distribución

También podemos representar la función de distribución:

curve(pnorm(x,170,12),xlim=c(130,210),col="blue",lwd=2,
      xlab="x",ylab="F(x)",main="Función de Distribución N(170,12)")

pnorm(180,170,12)

1-pnorm(168,170,12)

miDensidad=function(x) dnorm(x,170,12)

1.3 Representación del Area

La representación del área bajo una distribución normal se refiere al cálculo del área bajo la curva de la función de densidad de probabilidad de la distribución normal en un intervalo específico. Este área representa la probabilidad de que la variable aleatoria asociada con la distribución normal esté dentro de ese intervalo.

En términos más simples, la representación del área bajo una distribución normal nos ayuda a determinar la probabilidad de que una variable aleatoria caiga dentro de ciertos límites. Esto es especialmente útil en estadística y probabilidad para comprender la probabilidad de que un evento ocurra dentro de cierto rango, dado un conjunto de datos distribuidos normalmente.

A continuación podemos representar el area con los siguientes comandos:

regionX=seq(150,168,0.01)            # Intervalo a sombrear
xP <- c(150,regionX,168)             # Base para la creacion de poligonos que crean el efecto "sombra"
yP <- c(0,dnorm(regionX,170,12),0)   # Altura de los polígonos sombreados
curve(dnorm(x,170,12),xlim=c(130,210),yaxs="i",ylim=c(0,0.035),ylab="f(x)",
      main='Densidad N(170,12)') 
polygon(xP,yP,col="orange1")
box()

  • \(z_{0.95} = min {x : P ( X ≤ x ) ≥ 0.95}\)
qnorm(0.95,170,12)

1.4 Función que permite graficar el área bajo la curva

Tener presente que una función es un bloque de código que realiza una tarea específica cuando se llama o se invoca. Las funciones en RStudio pueden ser funciones predefinidas que vienen con el lenguaje R o pueden ser definidas por el usuario. Las funciones pueden aceptar entradas, realizar cálculos o manipulaciones de datos y producir salidas.

normal_area <- function(mean = 0, sd = 1, lb, ub, acolor = "lightgray", ...) {
    x <- seq(mean - 3 * sd, mean + 3 * sd, length = 100) 
    
    if (missing(lb)) {
       lb <- min(x)
    }
    if (missing(ub)) {
        ub <- max(x)
    }

    x2 <- seq(lb, ub, length = 100)    
    plot(x, dnorm(x, mean, sd), type = "n", ylab = "")
   
    y <- dnorm(x2, mean, sd)
    polygon(c(lb, x2, ub), c(0, y, 0), col = acolor)
    lines(x, dnorm(x, mean, sd), type = "l", ...)
}

LLamado de la funcion, se ingresa los parametros que se requieren en este caso:

  • mean es \(μ\), la media de la distribución.
  • sd es \(σ\) la desviacion desviación estándar.
  • lb es el limite inferior a calcular (variable aleatoria \(x_1\).)
  • ub es el limite superior a calcular (variable aleatoria \(x_2\).)
  • lwd es el grosor de la funcion de densidad.
  • acolor es el color de la parte sombreada.
normal_area(mean = 25, sd = 3, lb = 20, ub = 29, lwd = 4, acolor = "blue")
